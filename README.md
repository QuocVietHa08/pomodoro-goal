## Precondition

- NodeJS: v18
- Cocoapods: 1.12.1
- React native: 0.72.1

## Install

```
  git clone https://github.com/QuocVietHa08/promodoro-goal.git
  cd promodoro-goal
  npm run pod-install
  npm install
  npm run ios
```
